* ChatClientNET 프로젝트
- 채팅서버와 통신 가능한 .NET 버전용 클라이언트입니다.
- 빌드하기 위해서는 Visual Studio 2013 버전 중 C# 빌드가 가능한 버전이 필요합니다.

* ChatClientAndroid 프로젝트
- 채팅서버와 통신 가능한 안드로이드 모바일용 자바 클라이언트입니다.
- 빌드하기 위해서는 IDE인 Intellij 가 필요합니다.
- IntelliJ IDEA Community Edition 12.1.6 : http://www.jetbrains.com/idea/download/

*UnityServer 프로젝트
-유니티 클라와 연동되는 서버 프로젝트 입니다.
-컴파일 없이 테스트 해보시려면 BIN폴더의 exe파일을 CGSF\VSProject\Bin32\으로 옮겨주셔야 합니다.
-연결정보 127.0.0.1 : 25251

*UnityClient 프로젝트

-컴파일 없이 테스트 해보시려면 BIN폴더의 exe 파일을 이용해 주세요.
